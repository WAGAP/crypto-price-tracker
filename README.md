# Crypto Price Tracker

A simple tool that tracks live cryptocurrency prices using a public crypto market API.

## Features
- Track real-time prices for major cryptocurrencies
- Monitor BTC, ETH, and SOL price movements
- Display updated market price data
- Educational tool for learning how crypto market APIs work

## Technologies
- Python
- Crypto Market API (CoinGecko or similar)
- REST API integration
- JSON data processing

## Purpose
This project was created to explore cryptocurrency market data and understand how to retrieve and display real-time price information using public APIs.

## Future Improvements
- Support for more cryptocurrencies
- Price charts and visual analytics
- Real-time alerts for price changes
- Integration with portfolio tracking tools
